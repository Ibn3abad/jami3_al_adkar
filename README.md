# 📿 Jami3 Al Adkar

Ein moderner digitaler Begleiter für tägliche Adhkar (أذكار), Duas und islamische Erinnerungen.

## ✨ Überblick

Jami3 Al Adkar bietet eine benutzerfreundliche Plattform zum Lesen, Organisieren und Verfolgen täglicher islamischer Adhkar. Die Anwendung unterstützt Muslime dabei, ihre Morgen- und Abend-Adhkar sowie weitere authentische Bittgebete einfach in ihren Alltag zu integrieren.

## 🚀 Features

- 🌅 Morgen-Adhkar
- 🌙 Abend-Adhkar
- 🤲 Tägliche Duas
- 📖 Authentische Quellenangaben
- 🔍 Schnelle Suche
- ⭐ Favoritenfunktion
- 📱 Responsive Design
- 🌍 Mehrsprachige Unterstützung
- 🌙 Dark Mode
- 📊 Fortschrittsverfolgung
